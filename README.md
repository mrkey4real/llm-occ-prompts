# LLM-occ Prompts and Disclosure Materials

This repository provides the prompt templates and disclosure materials for the manuscript:

**“Harnessing Occupant Intention for Occupant-Centric Control with Large Language Models (LLMs) in Residential Buildings.”**

## What is included
- Prompt templates used in the study:
  - `prompts/observer.txt`
  - `prompts/review.txt`
  - `prompts/future.txt`
- Manuscript disclosure text:
  - `docs/data_availability_statement.md`
  - `docs/code_availability_statement.md`
- Paper results:
  - `gpt-5-2025-08-07`

## What is not included
- Raw camera videos and personally identifiable data.
- Private credentials and institution-restricted assets.

If you use these materials, please cite the corresponding paper (to be updated after publication).
